Technology

Built using Flask, Bootstrap, and SQLite3 for the database.

Python3 was used to model the data and send requests via Chess.com's API.

About ChessDB

ChessDB is a statistical chess website where users can download their games.

It also models your data in a way that users can see their stats in a meaningful way.
